## üé£ What is x3?

`x3` is a lightweight, open-source Discord bot built for community fun. It features:

* **Fishing**: Cast your line and catch random fish!
* **Economy**: Earn money, buy bait, upgrade your rods.
* **Shop System**: Buy bait and rods to improve your fishing odds.
* **Gambling Games**: Try your luck with `/cockfight`, or rob other users!
* **Leaderboards**: Compete to be the richest angler around.

Everything is saved locally using JSON ‚Äî simple and private.

## üõ† How It Works

* Built with `discord.py`
* Stores all user data in `data/users.json`
* Commands organized using `cogs/` for modularity
* MIT Licensed. Free to use, host, and modify

## üöÄ Self-Hosting

To host your own version of `x3`:

```bash
git clone https://github.com/myferr/x3
cd x3
pip install -r requirements.txt
# Add your bot token to a .env file
python src/bot.py
```
